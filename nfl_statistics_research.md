# NFL Statistics Research by Position

Based on the Pro Football Reference glossary, here are the comprehensive statistics tracked for each NFL position:

## Quarterback Statistics

### Quarterback Standard Metrics

- **Passing Yards (Yds)**: Total yards gained through passing
- **Passing Touchdowns (TD)**: Number of touchdown passes thrown
- **Interceptions (Int)**: Number of passes intercepted
- **Completions (Cmp)**: Number of completed passes
- **Passing Attempts (Att)**: Total pass attempts
- **Completion Percentage (Cmp%)**: Cmp/Att
- **Passer Rating (Rate)**: NFL passer rating formula
- **Times Sacked (Sk)**: Number of times sacked
- **Sack Percentage (Sk%)**: (Times sacked)/(passing attempts + sacks)
- **Touchdown Percentage (TD%)**: (Passing TD)/(Passing attempts)

### Quarterback Advanced Metrics

- **Yards per Attempt (Y/A)**: Passing yards / passing attempts
- **Adjusted Yards per Attempt (AY/A)**: (pass yards + 20*(pass TD) - 45*(interceptions))/(passing attempts)
- **Net Yards per Attempt (NY/A)**: (pass yards - sack yards)/(passing attempts + sacks)
- **Adjusted Net Yards per Attempt (ANY/A)**: (pass yards + 20*(pass TD) - 45*(interceptions) - sack yards)/(passing attempts + sacks)
- **Expected Points**: Estimated point value based on down, distance, field position

## Running Back Statistics

### Running Back Standard Metrics

- **Rushing Yards (Yds)**: Total yards gained rushing
- **Rushing Touchdowns (TD)**: Number of rushing touchdowns
- **Rushing Attempts (Att)**: Total rush attempts
- **Yards per Attempt (Y/A)**: Rushing yards / rushing attempts
- **Long Gain (Lng)**: Longest rush of the season
- **Fumbles (Fmb)**: Total fumbles
- **Fumbles Lost**: Fumbles not recovered by own team

### Receiving Metrics (for RBs who catch passes)

- **Receptions (Rec)**: Number of passes caught
- **Receiving Yards (Yds)**: Yards gained through receiving
- **Receiving Touchdowns (TD)**: Touchdowns scored through receiving
- **Yards per Reception (Y/R)**: Receiving yards / receptions
- **Targets**: Number of times targeted in passing game

### Combined Metrics

- **Yards from Scrimmage (YScm)**: Rushing yards + receiving yards
- **Total Touchdowns (TD)**: Rushing TDs + receiving TDs
- **Rushing + Receiving TDs (RRTD)**: Combined rushing and receiving touchdowns

## Wide Receiver Statistics

### Wide Receiver Standard Metrics

- **Receptions (Rec)**: Number of passes caught
- **Receiving Yards (Yds)**: Total yards gained receiving
- **Receiving Touchdowns (TD)**: Touchdowns scored through receiving
- **Targets**: Number of times targeted in passing game
- **Yards per Reception (Y/R)**: Receiving yards / receptions
- **Receptions per Game (R/G)**: Rec / games played
- **Long Reception (Lng)**: Longest reception of the season
- **Fumbles (Fmb)**: Total fumbles

### Wide Receiver Advanced Metrics

- **Catch Percentage**: Receptions / targets
- **Yards per Target**: Receiving yards / targets
- **Air Yards**: Yards the ball traveled in the air
- **Yards After Catch (YAC)**: Yards gained after catching the ball

## Tight End Statistics

### Tight End Standard Metrics

- **Receptions (Rec)**: Number of passes caught
- **Receiving Yards (Yds)**: Total yards gained receiving
- **Receiving Touchdowns (TD)**: Touchdowns scored through receiving
- **Targets**: Number of times targeted in passing game
- **Yards per Reception (Y/R)**: Receiving yards / receptions

### Blocking Metrics

- **Blocking Efficiency**: Effectiveness in run/pass blocking
- **Pancake Blocks**: Dominant blocks that knock defender to ground

## Offensive Line Statistics

### Offensive Line Standard Metrics

- **Sacks Allowed**: Number of sacks allowed
- **Quarterback Hits Allowed**: Number of hits on QB
- **Hurries Allowed**: Number of QB hurries allowed
- **Pancake Blocks**: Dominant blocks
- **Penalties**: Holding, false start, etc.

### Offensive Line Advanced Metrics

- **Pass Block Win Rate**: Percentage of pass blocks won
- **Run Block Win Rate**: Percentage of run blocks won
- **Pressure Rate Allowed**: Percentage of plays allowing pressure

## Defensive Line Statistics

### Defensive Line Standard Metrics

- **Tackles (Tkl)**: Solo + assisted tackles
- **Sacks (Sk)**: Tackling quarterback behind line of scrimmage
- **Tackles for Loss (TFL)**: Tackles behind line of scrimmage
- **Quarterback Hits (QB Hits)**: Hits on the quarterback
- **Forced Fumbles (FF)**: Fumbles caused by defensive player
- **Fumble Recoveries (FR)**: Fumbles recovered
- **Passes Defensed (PD)**: Passes broken up or knocked down

### Defensive Line Advanced Metrics

- **Pressure Rate**: Percentage of plays generating pressure
- **Run Stop Percentage**: Percentage of runs stopped for short gain/loss
- **Pass Rush Win Rate**: Percentage of pass rushes won
- **Tackle Efficiency**: Tackles per snap played

## Linebacker Statistics

### Linebacker Standard Metrics

- **Tackles (Tkl)**: Solo + assisted tackles
- **Sacks (Sk)**: Tackling quarterback behind line of scrimmage
- **Tackles for Loss (TFL)**: Tackles behind line of scrimmage
- **Interceptions (Int)**: Passes caught by defense
- **Passes Defensed (PD)**: Passes broken up
- **Forced Fumbles (FF)**: Fumbles caused
- **Fumble Recoveries (FR)**: Fumbles recovered
- **Quarterback Hits**: Hits on the quarterback

### Linebacker Advanced Metrics

- **Tackle Efficiency**: Tackles per snap played
- **Coverage Snaps**: Number of snaps in pass coverage
- **Run Defense Snaps**: Number of snaps in run defense
- **Blitz Rate**: Percentage of snaps blitzing
- **Completion Percentage Allowed**: Completion % when targeted

## Defensive Back Statistics

### Defensive Back Standard Metrics

- **Tackles (Tkl)**: Solo + assisted tackles
- **Interceptions (Int)**: Passes caught by defense
- **Passes Defensed (PD)**: Passes broken up or knocked down
- **Interception Return Yards**: Yards gained on interception returns
- **Interception Return TDs**: Touchdowns scored on interception returns
- **Forced Fumbles (FF)**: Fumbles caused
- **Fumble Recoveries (FR)**: Fumbles recovered

### Defensive Back Advanced Metrics

- **Target Rate**: Percentage of snaps targeted by opposing QBs
- **Completion Percentage Allowed**: Completion % when targeted
- **Yards per Target Allowed**: Yards allowed per target
- **Passer Rating Allowed**: Passer rating of quarterbacks when targeting this player
