## Context

**What is the issue or problem we're addressing?**

## Decision

**What change are we making?**

State the decision clearly and concisely. This should be:

- Specific and actionable

- Time-bound if relevant

## Rationale

**Why are we making this decision?**

Explain the reasoning behind the decision:

- Key factors that led to this choice
- Trade-offs considered
- Benefits expected
- Risks understood and accepted

## Consequences

**What becomes easier or more difficult because of this decision?**

- What problems does this solve?
- What capabilities does this enable?
- What improvements does this bring?

### Negative Consequences

- What limitations does this introduce?

- What technical debt might this create?

### Neutral Consequences

- What changes but isn't strictly positive or negative?
