### DL Fatigue Management

```python
def should_rotate_dl(player, rotation_state):
    # Force rotation conditions
    if rotation_state.consecutive_snaps(player) >= 6:
        return True  # Too many consecutive snaps

    if rotation_state.snap_percentage(player) >= 0.65:
        return True  # Approaching snap cap

    if rotation_state.fatigue(player) >= 60:
        return True  # Too fatigued

    return False
```
